# Using Codable (Encode & Decodable) in Swift



```swift
var arrayOfNames: String = []
arrayOfStrings.append("Ben")
arrayOfStrings.append("Ken")
arrayOfStrings.append("Len")
for name in arrayOfNames{
  print("\(name) ")
}
// prints 'Ben Ken Len' to the console.
```
